---Slide Puzzle game---
At the initial screen key in your name and select difficulty of puzzle
Game will render grid size according to difficulty
To use a different photo, slice up photo put it in its own folder with folder name as the number of photos inside
Number of photo must be in perfect square (ie. 4x4, 7x7)
Copy and replicate conditions in function newgame.
